# Iconos para Modo Claro/Oscuro

## Imágenes necesarias

Necesitás descargar dos imágenes y colocarlas en la carpeta `public/`:

1. **luz.png** - Icono para modo claro (sol/luz)
2. **oscuro.png** - Icono para modo oscuro (luna/noche)

## Recomendaciones Premium de Flaticon.com

### Para luz.png (Modo Claro):
1. **"Sun icon"** - Buscar: "sun" o "sunshine"
   - URL sugerida: https://www.flaticon.com/search?word=sun
   - Estilo: Premium, minimalista, con gradiente dorado/amarillo
   - Tamaño recomendado: 512x512px o 256x256px
   - Formato: PNG con fondo transparente

2. **"Light bulb icon"** - Buscar: "light bulb" o "lamp"
   - URL sugerida: https://www.flaticon.com/search?word=light%20bulb
   - Estilo: Premium, moderno, con efecto de brillo
   - Tamaño recomendado: 512x512px o 256x256px

### Para oscuro.png (Modo Oscuro):
1. **"Moon icon"** - Buscar: "moon" o "crescent moon"
   - URL sugerida: https://www.flaticon.com/search?word=moon
   - Estilo: Premium, minimalista, con gradiente azul/plateado
   - Tamaño recomendado: 512x512px o 256x256px
   - Formato: PNG con fondo transparente

2. **"Night mode icon"** - Buscar: "night mode" o "dark mode"
   - URL sugerida: https://www.flaticon.com/search?word=night%20mode
   - Estilo: Premium, moderno, con estrellas o efecto nocturno
   - Tamaño recomendado: 512x512px o 256x256px

## Búsquedas específicas recomendadas:

### En Flaticon.com:
1. Buscar: **"sun icon premium"** o **"sun icon gradient"**
2. Buscar: **"moon icon premium"** o **"crescent moon icon"**
3. Filtrar por: **Premium** y **PNG**
4. Tamaño: **512x512** o **256x256**

## Iconos específicos recomendados (Premium):

### Opción 1 - Minimalista:
- **Sol**: Buscar "sun icon flat" con estilo minimalista
- **Luna**: Buscar "crescent moon icon" con estilo minimalista

### Opción 2 - Con gradientes:
- **Sol**: Buscar "sun icon gradient" con gradiente amarillo/dorado
- **Luna**: Buscar "moon icon gradient" con gradiente azul/plateado

### Opción 3 - Moderno/3D:
- **Sol**: Buscar "sun icon 3d" o "sun icon modern"
- **Luna**: Buscar "moon icon 3d" o "moon icon modern"

## Instrucciones de descarga:

1. Ir a https://www.flaticon.com
2. Buscar los iconos mencionados arriba
3. Filtrar por "Premium" si tenés cuenta premium
4. Descargar en formato PNG
5. Tamaño: 512x512px o 256x256px
6. Renombrar como:
   - `luz.png` (para el sol/luz)
   - `oscuro.png` (para la luna/noche)
7. Colocar en la carpeta `frontend/public/`

## Nota:
Si no tenés cuenta premium, podés usar iconos gratuitos pero asegurate de que sean de alta calidad y con fondo transparente.
